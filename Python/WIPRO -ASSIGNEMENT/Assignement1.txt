P6: Write a program that takes a grade (A, B, C, D, F) as input and uses match-case
to print a message corresponding to the grade (e.g., "Excellent" for A, "Good" for
B, etc.).



grade = input("Please enter a grade (A, B, C, D, F): ").upper()
match grade:
    case 'A':
        print("Excellent")
    case 'B':
        print("Good")
    case 'C':
        print("Average")
    case 'D':
        print("Below Average")
    case 'F':
        print("Fail")
    case _:
        print("Invalid grade. Please enter A, B, C, D, or F.")

P11:  Write a program that uses a for loop to count the number of vowels in a given
string. Print the count.

vowel_count = 0
for char in string.lower():
    if char in 'aeiou':
        vowel_count += 1
print(f"The number of vowels in the string is: {vowel_count}.")
